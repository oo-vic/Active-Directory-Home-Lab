# Active-Directory-Home-Lab
A home lab built to simulate a real enterprise network environment using Active Directory. Built with VMware Workstation Pro, Windows Server 2022, and Windows 11 Pro as the client machine.   Based on the [East Charmer YouTube Series] (https://www.youtube.com/playlist?list=PLAdEnQWAAbfXMY2D4HVZOe-ChfTKmaJfQ)
